# Voice DNA — Extraction Prompt

**How to use this:**
1. Open Claude, ChatGPT, or any AI
2. Copy everything below the line
3. Paste it into a new chat
4. Dump all your writing underneath it (emails, posts, transcripts, messages — anything)
5. Hit send
6. Save the output as `yourname-voice-dna.md`

**Minimum:** ~1,500 words of your own writing. More = more accurate.
